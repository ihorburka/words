## 📝 Про проєкт [UA]
**Дата створення:** 19.10.2024

Цей проєкт був створений для вивчення англійських слів та тренування швидкості їх друку. Програма вийшла невеликою і, чесно кажучи, мало допомогла мені з самою англійською. Проте для мене вона особлива: це один із тих проєктів, що надихнули мене вивчати програмування та показали справжній сенс технологій.

Проєкт є архівним. Він більше не підтримується, не оновлюється і залишається тут таким, яким він є ("as is") — просто як пам'ять про мої кроки та мотивацію.

## 📝 About the Project [EN]
**Created:** October 19, 2024

This project was created to learn English words and practice typing them quickly. The program turned out to be quite small, and to be honest, it didn't help me much with English itself. However, it is special to me: it is one of those projects that inspired me to learn programming and gave real meaning to technology.

This is an archived project. It is no longer maintained or updated and is left here exactly as it is ("as is") — simply as a memory of my journey and motivation.
